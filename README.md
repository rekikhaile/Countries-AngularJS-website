# Countries-AngularJS-website
Website Description: List of some countries with some information
Structure Folder/file	=> Technology used	=> Description
Front-end/Countries.html		=> HTML5,AngularJS, bootstrap, jQuery	=> home page made with HTML5 and uses AngularJS framework in order to display the list of countries. The javascript that takes care of the request to the backend to list countries is JavaScriptCountries.js and JavaScriptaddCountry.js is used to add countries. The page uses bootstrap template for UI elements. The add and hide buttons show jQuery example usage.
Front-end/DatePicker.html	=> jQuery UI	=> This html page is added just as a JQuery UI example.
Front-end/maat.css		=> bootstrap,CSS3		=> Main CSS file edited from the bootstrap template to match the look we wanted for our page.
Front-end/css/ bootstrap.min.css		=> bootstrap 	=> 	Bootstrap core CSS
Front-end/js/JavaScriptaddCountry.js		=> HTML DOM , jQuery AJAX		=> using a jQuery script that will attach and define the AJAX functions using jQuery method.
Front-end/js/JavaScriptCountries.js		=> jQuery AJAX, AngularJS		=>  using Angular script that will create a module + controller and attaches and defines the AJAX functions using jQuery method.
img			=> 	=> All the images used for our site are located in this folder.
Front-end/Back-end/servercountries/Info/countries.json		=> 	=> JSON	 File with JSON data format which has country information.
Front-end/Back-en/servercountries/servercountries		=> Node.js		=> The back-end for this project.
